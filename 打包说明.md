# Fenxi8 打包说明

## 打包完成

项目已成功打包为Windows可执行文件（.exe）！

### 生成的文件

在 `dist/` 目录下生成了两个版本：

1. **Fenxi8.exe** (69MB) - GUI图形界面版本
   - 双击即可运行
   - 提供友好的可视化操作界面
   - 适合日常使用

2. **Fenxi8_CLI.exe** (66MB) - CLI命令行版本
   - 在命令行中运行
   - 适合自动化脚本和定时任务
   - 支持命令行参数

## 使用方法

### 方法一：GUI版本（推荐）

1. 将以下文件和文件夹复制到任意位置：

   ```
   dist/Fenxi8.exe      # 主程序
   ```

2. 在同一目录下创建文件夹结构：

   ```
   你的文件夹/
   ├── Fenxi8.exe       # 复制过来的程序
   ├── FTdata/          # 放置首测CSV文件
   │   └── *.csv
   └── RTdata/          # 放置终测CSV文件
       └── *.csv
   ```

3. 双击 `Fenxi8.exe` 运行

4. 在界面中：
   - 选择FTdata文件夹
   - 选择RTdata文件夹
   - 选择输出目录
   - 点击"开始分析"

5. 分析完成后，在输出目录会生成：
   - `report.html` - 交互式图表报告
   - `assets/` - ECharts库文件夹（自动生成）
   - `fenxi8_*.log` - 日志文件

### 方法二：CLI版本

1. 准备文件夹结构（同上）

2. 在命令行中运行：

   ```bash
   Fenxi8_CLI.exe
   ```

3. 或者指定输出目录：

   ```bash
   Fenxi8_CLI.exe "D:\output"
   ```

4. 查看生成的 `report.html`

## ECharts库说明

### 自动处理机制

程序已内置智能ECharts库管理：

1. **自动生成**：每次生成HTML报告时，程序会自动将 `assets/` 文件夹复制到输出目录

2. **离线可用**：生成的HTML报告使用本地ECharts库，无需网络连接

3. **路径自适应**：
   - 开发环境：从源代码目录读取
   - 打包后：从exe的临时解压目录读取

### 文件结构

```
输出目录/
├── report.html          # HTML报告
└── assets/              # ECharts库（自动生成）
    └── echarts.min.js   # ECharts核心库
```

### 重要提示

⚠️ **不要删除assets文件夹**：

- HTML报告依赖 `assets/echarts.min.js` 才能正常显示图表
- 如果删除，图表将无法渲染
- 重新运行程序可再次生成

## 常见问题

### Q1: 运行exe提示找不到模块？

**A**: 不应该出现此问题。PyInstaller已将所有依赖打包到exe中。

### Q2: HTML报告打开后图表不显示？

**A**: 检查以下几点：

1. 确认输出目录有 `assets/echarts.min.js` 文件
2. 用浏览器打开HTML文件（推荐Chrome、Edge）
3. 按F12查看控制台是否有错误信息

### Q3: 如何重新生成assets文件夹？

**A**: 重新运行一次分析程序即可自动生成。

### Q4: exe文件太大（60+MB）？

**A**: 这是正常的，因为包含了：

- Python解释器
- pandas、numpy等科学计算库
- tkinter图形界面库
- 所有依赖项

如需减小体积，可以：

1. 使用UPX压缩（已启用）
2. 移除不必要的库
3. 使用虚拟环境只安装必需的包

### Q5: 能否在没有Python的电脑上运行？

**A**: 可以！打包后的exe是独立的可执行文件，不需要安装Python。

## 技术细节

### PyInstaller配置

关键配置项：

```python
datas=[
    ('assets', 'assets'),  # 包含ECharts库
    ('README.md', '.'),    # 包含帮助文档
]
```

### 资源文件定位

程序使用以下代码智能定位资源：

```python
if getattr(sys, 'frozen', False):
    # 打包后的exe环境
    base_path = sys._MEIPASS
else:
    # 开发环境
    base_path = os.path.dirname(os.path.abspath(__file__))
```

### 清理缓存

如果需要重新打包，先清理缓存：

```bash
pyinstaller --clean Fenxi8.spec
```

或删除以下目录：

- `build/`
- `dist/`
- `__pycache__/`
